# 3D Abstract Templates

Prompts for CGI-style renders that look like Blender, Cinema 4D, Houdini output.

## Key Characteristics
- Clean, controlled lighting
- Defined material properties
- Studio or void backgrounds
- Intentional, designed compositions
- Technical render quality

## Dual-Tone Lighting Setups

| Combination | Prompt Fragment |
|-------------|-----------------|
| Purple + Orange | cool purple key light from left, warm orange rim light from right |
| Cyan + Magenta | cyan key light, magenta accent light, neon aesthetic |
| Blue + Gold | cool blue main lighting, warm golden accent highlights |
| Teal + Coral | teal key light, coral warm accents |
| Monochrome + Accent | white/grey lighting, single [color] accent rim light |
